# Banco API – Proyecto Fullstack

## Resumen del aplicativo
Banco API es una solución que permite gestionar información bancaria básica.  
El aplicativo permite

- Registrar y administrar clientes (personas que acceden al banco).
- Crear y administrar cuentas bancarias asociadas a los clientes.
- Registrar movimientos (débitos y créditos) sobre cada cuenta, actualizando el saldo.
- Consultar un reporte de movimientos por cuenta, incluyendo saldo actualizado y detalle de transacciones.
- Descargar el reporte en formato PDF.
- Realizar búsquedas rápidas de registros de clientes, cuentas o movimientos.

## Qué permite hacer 
------
 Clientes  Crear, editar, listar y eliminar clientes 
 Cuentas  Crear, editar, listar y eliminar cuentas bancarias 
 Movimientos  Registrar débitoscréditos y consultar historial por cuenta 
 Reportes  Consultar y descargar reporte de movimientos por cuenta en PDF 

El aplicativo muestra mensajes de validación y error cuando alguna acción no cumple las reglas del banco.
